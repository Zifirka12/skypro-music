# Skypro Music 🎵 №3

Музыкальный сервис, созданный с использованием Next.js, TypeScript и Redux Toolkit.

## 🚀 Основные возможности

### ✅ Авторизация и регистрация

- Регистрация новых пользователей (`/signup`)
- Вход в систему (`/signin`)
- Автоматическое восстановление сессии из Local Storage
- Защита маршрутов (редирект неавторизованных пользователей)

### ✅ Просмотр и поиск треков

- Главная страница с полным каталогом треков
- Страницы подборок (Плейлист дня, 100 танцевальных хитов, Инди-заряд)
- Поиск треков по названию (частичное совпадение, без учета регистра)
- Фильтрация и сортировка треков

### ✅ Избранное

- Добавление/удаление треков в избранное (лайк)
- Страница "Мой плейлист" (`/my-playlist`) с избранными треками
- Синхронизация избранного с сервером
- Сохранение состояния избранного

### ✅ Аудио плеер

- Воспроизведение треков
- Управление воспроизведением (play/pause, предыдущий/следующий трек)
- Регулировка громкости
- Прогресс-бар с возможностью перемотки
- Режим повтора (repeat)
- Режим перемешивания (shuffle)
- Плеер скрыт по умолчанию и появляется при выборе трека
- Состояние плеера не сохраняется при перезагрузке страницы

### ✅ Дополнительные страницы

- Страница 404 (`/not-found`) с кастомным дизайном
- Навигация между страницами

## 🛠 Технологии

- **Next.js 15.5.4** - React фреймворк с App Router
- **TypeScript** - типизация кода
- **Redux Toolkit** - управление состоянием приложения
- **React Redux** - интеграция Redux с React
- **CSS Modules** - изоляция стилей компонентов
- **classnames** - работа с условными классами
- **ESLint & Prettier** - качество и форматирование кода

## 📦 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка проекта для production
npm run build

# Запуск production сборки
npm start

# Проверка кода линтером
npm run lint

# Форматирование кода
npm run format
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

- Автоматическое восстановление сессии

Приложение использует REST API для:

- Авторизации и регистрации пользователей
- Получения списка треков
- Получения подборок
- Работы с избранными треками (лайки)

## 🚀 Деплой

Проект задеплоен на [Vercel](https://vercel.com) - бесплатная платформа для Next.js приложений.

### Как задеплоить:

1. Зарегистрируйтесь на [vercel.com](https://vercel.com) через GitHub
2. Нажмите "Add New Project" и выберите ваш репозиторий
3. Vercel автоматически определит Next.js и настроит всё необходимое
4. Нажмите "Deploy" и дождитесь завершения сборки
5. Готово! Ваш проект будет доступен по адресу `https://your-project.vercel.app`

После каждого push в ветку `main` будет происходить автоматический деплой.
